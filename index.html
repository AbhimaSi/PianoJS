PianoJS: Simulador de Teclado

Olá! bem vindo ao PianoJS, um simulador simples de piano.
Este projeto é feito com o propósito de estudar as abordagens de programação na linguagem JavaScript.

Como funciona?

Você pode utilizar seu teclado ou o seu mouse para tocar notas utilizando as teclas do piano!
